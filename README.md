# Aula_JavaScript
Aulas e projetos JavaScript para treino.

### Aula 05

- **DOM** = Document object model;

- **Variavel** = é um espaço reservado na memória do computador usado para armazenar dados ou valores. Ela funciona como uma "caixa" rotulada com um nome único (identificador), permitindo que você recupere, altere e manipule essas informações ao longo da execução do programa;

- **const** = indica que a variavel é constante;

- **let** = indica que a variavel que armazena referencias;
